 EVENT MANAGEMENT

 We created a event management system using JAVA and MYSQL as database.
 
 1.Firstly, import through build path 4 jar files as preferred libraries.
 2.Then you need to import all the uploaded database dump files into your MYSQL . 
 3.Save your databse table as- "event_managment".
 4.Then  in DBConnect class in login package you have to enter your MYSQL id and password. 
 5.Then to show email you have to enter sender gmail address and password in fromUser and fromUserPassword respectively.
    And we have used google server for email .
 6.In sender gmail he has to update two settings- Two step verification should be turned off and Allow Less Secure App should be turned on.
